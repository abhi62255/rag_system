# Amazon Smart Thermostat – Technical Specifications

## Physical Dimensions
| Property | Value |
|----------|-------|
| Width | 3.56 inches |
| Height | 3.56 inches |
| Depth | 0.84 inches |
| Color | White |

## Display & Interface
- Soft touchscreen faceplate
- Color icons: Red (heat), Blue (AC)
- Digital temperature number display
- Buttons: Temperature Up/Down, Select/Mode

## Connectivity
- Wi-Fi: 2.4 GHz (router must have 2.4 GHz enabled)
- Smart Home Platform: Amazon Alexa
- No built-in speaker or microphone (voice control via separate Echo device or Alexa app)

## Compatibility
- Works with most 24V HVAC systems
- Compatible with select Echo devices as external temperature sensors
- Compatible with Amazon Smart Air Quality Monitor as temperature sensor

## Certifications
